# 🖥️ 호텔 예약 시스템 화면 구조

## 전체 화면 구성

### 사용자 화면 (일반 사용자)

```
┌─────────────────────────────────────┐
│         공통 레이아웃 (Layout)       │
│  ┌───────────────────────────────┐ │
│  │  Header: 로고, 메뉴, 로그인    │ │
│  └───────────────────────────────┘ │
│  ┌───────────────────────────────┐ │
│  │         페이지 컨텐츠          │ │
│  └───────────────────────────────┘ │
│  ┌───────────────────────────────┐ │
│  │      Footer: 연락처, 정보      │ │
│  └───────────────────────────────┘ │
└─────────────────────────────────────┘
```

## 페이지별 상세 구조

### 1. 홈 (Home)
**경로**: `/`

```
┌─────────────────────────────────────┐
│         환영 메시지 (Hero)           │
│  - 사용자 이름 표시 (로그인 시)      │
│  - 객실 조회 버튼                    │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         공지사항 목록                │
│  - 최근 5개 공지사항                 │
│  - 중요 공지 상단 표시               │
│  - 더보기 버튼                       │
└─────────────────────────────────────┘
```

**주요 기능**:
- 로그인 상태에 따른 환영 메시지
- 공지사항 미리보기
- 빠른 객실 조회 링크

---

### 2. 로그인 (Login)
**경로**: `/login`

```
┌─────────────────────────────────────┐
│         로그인 폼                    │
│  - 아이디 입력                       │
│  - 비밀번호 입력                     │
│  - 로그인 버튼                       │
│  - 아이디/비밀번호 찾기 링크         │
│  - 회원가입 링크                     │
└─────────────────────────────────────┘
```

**상태 관리**:
- 로그인 성공 → JWT 토큰 저장 → 홈으로 이동
- 로그인 실패 → 에러 메시지 표시

---

### 3. 회원가입 (Register)
**경로**: `/register`

```
┌─────────────────────────────────────┐
│         회원가입 폼                  │
│  - 아이디 입력                       │
│  - 비밀번호 입력                     │
│  - 비밀번호 확인                     │
│  - 이메일 입력                       │
│  - 이메일 인증 코드 입력             │
│  - 인증 코드 전송 버튼               │
│  - 닉네임 입력                       │
│  - 회원가입 버튼                     │
└─────────────────────────────────────┘
```

**유효성 검사**:
- 아이디 중복 체크
- 이메일 인증 필수
- 비밀번호 일치 확인

---

### 4. 객실 조회 (Rooms)
**경로**: `/rooms`

```
┌─────────────────────────────────────┐
│         필터 섹션                    │
│  - 체크인 날짜 선택                  │
│  - 체크아웃 날짜 선택                │
│  - 검색 버튼                         │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         객실 목록 (Grid)             │
│  ┌──────┐  ┌──────┐  ┌──────┐      │
│  │ 이미지│  │ 이미지│  │ 이미지│      │
│  │ 객실명│  │ 객실명│  │ 객실명│      │
│  │ 가격  │  │ 가격  │  │ 가격  │      │
│  │ 상세보기│ │ 상세보기│ │ 상세보기│ │
│  └──────┘  └──────┘  └──────┘      │
└─────────────────────────────────────┘
```

**객실 카드 정보**:
- 객실 이미지
- 객실명, 타입
- 1박 가격
- 예약 가능 여부 (날짜 필터 적용 시)

**상세 모달**:
- 객실 상세 정보
- 리뷰 목록
- 같은 타입의 다른 뷰 선택
- 예약하기 버튼

---

### 5. 예약 (Booking)
**경로**: `/booking?roomId=1&checkIn=2025-01-15&checkOut=2025-01-17`

```
┌─────────────────────────────────────┐
│         예약 정보 요약               │
│  - 객실명, 설명                      │
│  - 체크인/체크아웃 날짜 (읽기 전용) │
│  - 숙박 일수 계산                    │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         예약 상세 정보               │
│  - 인원수 선택                       │
│  - 특별 요청사항 입력                │
│  - 총 금액 계산 표시                 │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         결제 방법 선택               │
│  ○ 카드                             │
│  ○ 계좌이체                         │
│  ○ 현금                             │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         예약하기 버튼                │
└─────────────────────────────────────┘
```

**계산 로직**:
- 총 금액 = 객실 가격 × 숙박 일수
- 숙박 일수 = 체크아웃 - 체크인

---

### 6. 마이페이지 (MyPage)
**경로**: `/mypage`

```
┌─────────────────────────────────────┐
│         탭 메뉴                      │
│  [예약 내역] [리뷰] [프로필]        │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         예약 내역 탭                 │
│  - 예약 목록 (카드 형태)             │
│    - 객실명, 날짜, 상태              │
│    - 취소 버튼 (CONFIRMED 상태만)   │
│  - 예약 상세 모달                    │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         리뷰 탭                      │
│  - 작성한 리뷰 목록                  │
│  - 리뷰 작성 버튼 (예약 완료 후)    │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         프로필 탭                    │
│  - 개인정보 수정 폼                  │
│  - 비밀번호 변경                     │
└─────────────────────────────────────┘
```

## 관리자 화면

### 관리자 대시보드 (AdminDashboard)
**경로**: `/admin`

```
┌─────────────────────────────────────┐
│         상단 메뉴                    │
│  [대시보드] [객실] [예약] [리뷰]    │
│  [공지사항] [통계] [프로필]         │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         대시보드 탭                  │
│  ┌──────────┐  ┌──────────┐        │
│  │ 예약     │  │ 청소완료  │        │
│  │  15      │  │  20      │        │
│  └──────────┘  └──────────┘        │
│  ┌──────────┐  ┌──────────┐        │
│  │ 청소필요 │  │ 보수중    │        │
│  │  5       │  │  2       │        │
│  └──────────┘  └──────────┘        │
│  ┌──────────────────────────────┐  │
│  │ 오늘 체크인 목록              │  │
│  │ - 예약 ID, 객실명, 고객명     │  │
│  │ - 체크인 버튼                 │  │
│  └──────────────────────────────┘  │
│  ┌──────────────────────────────┐  │
│  │ 오늘 체크아웃 목록            │  │
│  │ - 예약 ID, 객실명, 고객명     │  │
│  │ - 체크아웃 버튼               │  │
│  └──────────────────────────────┘  │
└─────────────────────────────────────┘
```

**대시보드 통계 카드**:
- 예약 (CONFIRMED + 미래 체크인)
- 청소 완료 객실 (CLEAN 상태)
- 청소 필요 (DIRTY 상태)
- 보수 중 (MAINTENANCE 상태)
- 오늘 체크인/체크아웃 수
- 현재 투숙 중 수

**객실 관리 탭**:
- 객실 목록 (테이블)
- 객실 추가/수정/삭제
- 객실 상태 변경 (CLEAN/DIRTY/MAINTENANCE)
- 객실 활성화/비활성화

**예약 관리 탭**:
- 예약 목록 (필터: 전체/CONFIRMED/CHECKED_IN 등)
- 예약 상세 정보
- 상태 변경 (체크인/체크아웃)

**리뷰 관리 탭**:
- 리뷰 목록
- 공개/비공개 설정
- 관리자 답변 작성

**통계 탭**:
- 월별 통계 그래프
- 연도별 통계 분석
- 매출 통계

## 컴포넌트 구조

### 공통 컴포넌트

```
components/
├── Layout.jsx          # 공통 레이아웃 (Header, Footer)
└── (추가 컴포넌트)
```

### 페이지 컴포넌트

```
pages/
├── Home.jsx
├── Login.jsx
├── Register.jsx
├── Rooms.jsx
├── Booking.jsx
├── MyPage.jsx
├── Notices.jsx
└── AdminDashboard.jsx
```

### 상태 관리

```
context/
└── AuthContext.jsx     # 인증 상태 관리 (JWT 토큰, 사용자 정보)
```

## 반응형 디자인

- **모바일**: 1열 그리드, 햄버거 메뉴
- **태블릿**: 2열 그리드
- **데스크톱**: 3-4열 그리드, 사이드바 메뉴

**Tailwind CSS**를 사용하여 반응형 디자인 구현
